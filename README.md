# wsl-dashboard

Управляй WSL-дистрибутивами через браузер. Запускай, останавливай,
смотри статистику использования — всё в одном окне.

Весит ~10 МБ RAM, работает как локальный сервер.

## Запуск

```bash
cargo run --release
# → открывает http://localhost:7070
```

Или просто:
```bash
cargo install --path .
wsl-dashboard
```

## Что умеет

- Список всех WSL-дистрибутивов с версией и статусом
- Запуск / остановка одной командой
- CPU и RAM каждого дистрибутива в реальном времени
- Открытие оболочки прямо из интерфейса
- Экспорт дистрибутива в .tar одной кнопкой
- Авто-обновление статуса каждые 2 секунды
