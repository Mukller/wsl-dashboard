# Code of Conduct

Коротко: не ведите себя как придурок.

Развёрнуто:

- Критикуйте код, не людей.
- Вопросы задавайте конкретно — что делали, что ожидали, что получили.
- Если нашли баг — покажите минимальный воспроизводящий пример.
- Pull request без описания — зря потраченное время для всех.

Нарушения: первый раз — предупреждение. Второй — бан.
